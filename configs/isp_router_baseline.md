# ISP Router Baseline Hardening

## Actions Taken
- Changed admin password
- Disabled WPS
- Disabled UPnP
- Disabled remote management
- Enabled WPA2/WPA3 wireless security
- Configured guest Wi-Fi separation

## Security Rationale
These steps reduce the router attack surface and protect against
unauthorised access and automatic port exposure.

- Edge device hardening
